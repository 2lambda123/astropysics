
Ccd
===

The ``ccd`` module contains classes and functions for processing CCD images or similar.  Detailed analysis tools are in ``astropysics.phot`` and ``astropysics.spec`` -- this module is mainly for low-level operations like direct viewing of images, calibrations, and similar.
