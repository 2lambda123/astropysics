
Plotting
========

The ``plotting`` module contains classes and funtions to aid in making plots.  These are mostly centered around `matplotlib <http://matplotlib.sourceforge.net/>`_, although some 3d plots instead use `mayavi <http://code.enthought.com/projects/mayavi/>`_.
